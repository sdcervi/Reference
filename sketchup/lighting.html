Lighting & Shadows

  • Home

If you want shadows that are a bit more real-world and dynamic than the default shadows, you can use the Shadows palette to control that. You can access that either through the graphical toolbar or the palette menu on the right of the window. Real-world shadows are calculated based on a few factors:

  • The model’s geolocation, if specified
  • The model’s compass orientation
  • The selected time zone
  • The specified time and date in the Shadows palette

Note: Shadows are not corrected for dayling saving time.

Palette options

Option Usage
Display Shadows button Toggles dymanically generated shadows on and off
(in the upper left corner)
Time Zone Selects the time zone or allows changes to the existing time zone. If the model is geolocated, this should be accurate.
Light Changes the intensity of highlights on illuminated surfaces
Dark Changes the intensity of shadows
Use Sun for Shading The simulated sun shades part of the model even when shadows are toggled off
On faces Faces can cast shadows on other faces. Note that this requires more graphics processing power, especially for larger and more complex models.
On ground Entities can cast shadows on the ground plane
From edges Edges not associated with a face can cast a shadow (off by default)

Faces with opacity less than 70% will not cast a shadow. If you want a face to cast a shadow, set the face’s opacity to 70% or higher.
