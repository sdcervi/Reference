The Asset Editor

This is the main hub for everything V-Ray. Use this to work with materials and adjust their properties, define and adjust lighting, change render settings, and more.

Settings

The settings panel includes these tabs:

  • Render: change which rendering engine is used (CPU or GPU/CUDA), the output quality, and whether to use a denoiser filter. Higher quality settings will take longer to render, especially on computers with lower specs. Progressive toggles the render method between progressive and bucket; progressive renders the image in passes, and bucket renders the image in chunks. Note: if your render is going to take a long time, set the Save image property to export the final image once the render is complete.
  • Camera: change camera settings and view properties
  • Render Output: change the dimensions and aspect ratio of your renders, and turn the Safe Frame on and off (this applies an overlay to help you compose your renders by showing you what will be within the frame)
  • Material Override: allows you to toggle a “clay model” view on and off; this replaces all materials with a flat neutral texture and can make adjusting lighting eaiser especially when working with surfaces with different textures, glossiness, and transparency. You can toggle which materials are immune to the override in their material properties. Making glass immune to override is usually a good idea.

Proxy objects

Proxies can be used for high-poly assets that would otherwise slow down a model’s performance (example: trees, couches, chairs with plus cushions). Creating a proxy essentially creates a low-poly placeholder within the SketchUp file that V-Ray then replaces with the high-poly asset at render time.

Note: the link file that V-Ray creates needs to retain its name and file location to work correctly. If it is moved or renamed, you’ll need to re-link it through the file path menu.

Materials

Using built-in V-Ray materials

As of October 2020, V-Ray 6 uses the Chaos interface for its material library. Open this with the Chaos button in the V-Ray toolbar, find the material you want, then import it into your model.

Once a material is imported:

  • Adjust its properties with the right-hand pane in the Asset Editor
  • Replace an existing material by right-clicking on the new material, selecting Use as replacement, then right-clicking on the target material and selecting Replace in scene
  • Apply to entities by selecting the entities, then right-clicking the new material and selecting Apply to selection
  • Apply to tags by making sure your model is tagged correctly (usually by material), right-clicking on the material in the Asset Editor, then selecting Apply to tag

Fur

V-Ray has a Fur tool to create fur-like or grass-like textures by applying a mesh without adding more geometry to the model itself. Fur only applies at render time and therefore doesn’t add any processing load to the SketchUp model.

Click here for more information about the Fur tool.

Organization

When working with materials, especially in large or complicated models, it’s a good idea to name all your materials according to what they are. For example, Conference table veneer, Conference table legs plastic, and Conference table legs steel. Since materials can’t be grouped into folders and are sorted alphabetically, name them so they are sorted in a way that makes sense.

You really don’t want to have eighteen instances of Color_001 and have no idea what each applies to.

Properties & parameters

Use the right pane to adjust materials. Select the material, then adjust the following options:

Diffuse

Use this to adjust the base color of a material, and/or the texture applied to it. Some textures are opaque and changing the base color won’t have any effect, but most built-in V-Ray textures are transparent and apply over the base color for easy adjustment.

Reflection

  • Reflection Color: this sets the highlight color within the reflection in the surface. Shadows will be the opposite color (example: red highlights, green shadows). Use a neutral gray for color-accurate reflections. The luminosity of the color affects how reflective the surface is.
  • Reflection Glossiness: how sharp reflections are. 1 is perfectly sharp reflections, lower numbers get blurrier the closer they get to 0.
  • Fresnel: changes reflectiveness based on viewing angle. This enables surfaces to be more reflective the further the camera gets from perpendicular to the surface.
  • Reflection IOR: changes how the reflected image displays on the surface
  • Metalness: sets how the surface behaves from a scale of 0 (non-conductive and non-metal) to 1 (conductive, metal). Basically, this affects diffusion of reflections on surfaces. Lower numbers will have smoothed, blurred reflections.
  • Surface Control: changes how the glossiness properties apply to the surface (i.e. whether Glossy or Rough evaluates to 1)

Refraction

This affects only transparent materials.

  • Refraction Color: changes what color light refracting through the surface is. Set to black to disable. If set to red, the material will look as if it only filters red light through. White or neutral gray will allow for color-accurate refraction. The luminosity of the color affects how refractive the surface is.
  • IOR: Index of Refraction. This affects how light bends when traveling through the surface. A value of 1 means light will not bend and the surface will function like a transparent window.
  • Refraction Glossiness: how sharp refractions are. 1 is perfectly sharp refractions (transparent window glass), lower numbers get blurrier the closer they get to 0 (frosted glass).
  • Thin-Walled: allows for single-surface refractive materials such as soap bubbles, curtains, or leaves.
  • Fog Color: Thicker objects are often less transparent than thin objects; this allows for that “fog” as an object thins or thickens.
  • Depth (cm): The depth at which fog begins to take effect. Higher values mean more translucency, lower values mean less translucency.
  • Translucency: sets how translucency works in the material. Volumetric is for fully translucent materials like liquids and glass; SSS enables non-transparent material scattering for materials such as skin, wax, marble, and other mostly-opaque materials.

Opacity

This sets the overall transparency of any entities with this material applied. Note that this is different from transparency/translucency with materials such as glass. One use case might be to place a semi-transparent object in the scene to suggest how an environment might be used.

Bump

A bump map allows you to apply depth to a surface texture, such as for wood. To create a bump map for a texture, desaturate the surface texture, boost the contrast and brightness, and then save as a separate file to apply as the map via the Bitmap option. This works best when a material’s dark areas are in relief and bright areas protrude, such as with wood planking. Adjust the strength with Amount.

Binding

Binds paramters to each other. For example, changing the base V-Ray material color will change the application color in SketchUp.

Removing unused materials

Use Window > Model Info > Statistics > Purge Unused (the button at the bottom) to remove any unused materials from your model to reduce processing load.

V-Ray components

The ground plane

If your model isn’t placed on top of other geometry, you can insert an infinite ground plane from the V-Ray toolbar. Move it so that it’s aligned with the bottom of your model (or at the desired offset), then apply a color or material to it. The color or material won’t show in the SketchUp interface, but will be visible in the rendering buffer. This is most useful when you have a standalone object and want a surface for it to reflect and cast shadows on.

One excellent use of the infite plane is to generate grass or carpet using the Fur tool, or a natural flat surface like sand or dirt. You may wish to increase the size of a ground plane’s texture to help reduce visible tiling.

Lighting

Adding lights to the model using the various V-Ray tools helps to make your renders look more realistic and finished.

Note: any copies of an existing light will be treated as instances of the same light (just like SketchUp components), and all of them will be affected by changes in the Asset Editor. For unique lights, create new light entities or right-click and select Make unique.

Ambient occlusion

This calculates how much each point in the scene is exposed to ambient lighting, and can create more realistic lighting easily. Toggle this on or off in the Settings tab of the Asset Editor, in the right parameters pane. It’s only available in CPU render mode. The difference is subtle but adds a lot. Pair with a denoiser for best results.

Sunlight

By default, SketchUp’s sunlight renders as crisp, direct, and bright. To change this, go to the Lighting tab in the Asset Editor, select the Sunlight element, then expand the right pane.

  • Color: controls the color of the light source. Use warmer colors for a sunrise or sunset, or cooler colors for a more stormy look.
  • Intensity: controls the brightness of the light source. Turning this up too much will blow out the image and cause it to look over-exposed.
  • Size Multiplier: controls the softness of the shadows. 0 is crisp edges; higher numbers mean blurrier/softer shadow edges.

Dome lights and HDRIs

Dome lights scatter light inward at the scene as if from a hemispherical dome over the scene. Dome lights are ideal for applying HDRI (High Dynamic Range Image) textures. You place the center of the dome in the model, then rotate it laterally to align the HDRI lighting with your renders’s lighting needs (make sure to enable the Use Transform option).

Note: most HDRIs render a bit dark and require increasing the intensity to look natural.

Sphere lights

These scatter light in all directions outward from the light source. Use sphere lights for lamp posts, light bulbs, or supplemental environmental lighting to illuminate features that are left a bit too dark by the model’s lights. Make sure to place the sphere lights so that they aren’t half inside, half outside of the model or component, since that can cause unexpected results.

Sphere lights will show in renders as large discs, unless set to Invisible and with Affect Reflections disabled (the Options tab of the parameters pane).

Omni lights

These are basically sphere lights, but with a softer glow and slightly better shadows. They&rsqou;re also invisible by default. Use them when sphere lights would be too harsh.

Rectangle light

Sphere lights might not cast enough light into a scene, so you can use rectangle lights to supplement them. These cast light in a single direction, indicated by an arrow in the SketchUp component.

Fog (volumetric environment)

Fog can be used to create atmospheric perspective, and make background elements less obtrusive. This also creates more depth in the image to add drama and visual interest.

Camera

SketchUp’s default field of view is 35mm, like a typical point-and-shoot camera. The human eye, however, has a field of view of usually 40–60mm; 50mm is a safe bet. Change this via View > Field of View, then type in your desired measurement.

For easy camera placement, use guides or lines to define the position and height of the camera. Anywhere between 5'6" and 5'10" should give a natural-feeling height to the camera, but if you know your intended audience is shorter or taller on average, you might want to adjust the camera height accordingly.

Use the Position Camera tool to place the camera for the scene. Click on your reference point for camera placement, then drag in the direction you want the camera to face.

Once you’re happy with the camera placement and direction, save the scene through the Scenes palette. This should also create a tab at the top for easy movement between scenes.

You can also use scenes to create different lighting scenarios for the same camera placement. Organize lighting elements by tags, then toggle tag visibility on and off for each scene as necessary.

Rendering

There are two types of renders that V-Ray can do:

  • Interactive: a lower-quality option that updates in real-time to reflect changes made in SketchUp and the V-Ray Asset Editor. Perfect for testing out different view angles and material properties.
  • Final: a higher-quality option best used for final renders for exporting. Does not update in realtime.

You’ll need to balance the rendering time against the quality needed. Higher quality renders will require longer to process.
