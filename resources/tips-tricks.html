Jump lines

Jump lines allow articles to span multiple non-consecutive pages without losing readers mid-article.

Example: ...continued on Page 6 and ...continued from Page 4.

Here‘s how to add dynamically updating jump lines in InDesign, that will use the correct page numbers even when an article is moved to a different page:

  1. At the end of the article’s copy on the page preceding the jump, create a new text frame that overlaps the article’s text frame and is aligned with its bottom, left, and right edges. Add a text wrap to the new frame so that it pushes the article copy out of the way.
  2. In this new text frame, type “Continued on page ” or your desired text that will come before the page number. Right-click, and choose Insert Special Character » Markers » Next Page Number. This will automatically add the page number for the next page in the article.
  3. Go to the next page in the article and add a new text frame, overlapping and with text wrap like the first one, on top of the first frame of the article.
  4. In this text frame, type “Continued from page ” or your desired text. Right-click, and choose Insert Special Character » Markers » Previous Page Number. This will automatically add the page number that the story continues from.

Lining up the jump lines is easier if the document has a baseline grid enabled.
