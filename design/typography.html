Terminology

Term Definition
character Digitally represents natural language. Each character can have several glyphs.
glyph Variations in how to draw a specific character. For example: a a a a a.
typeface A set of glyphs that share a common design. Typefaces are designed by one or multiple type designers. Type designers can form a type foundry.
foundry A company that designs typefaces.
styles Italic, bold, condensed, etc.
font The delivery format for one or more typeface styles. For example, Helvetica is a typeface, but helvetica-regular.otf is a font.
font family A collection of fonts with the same name and multiple styles. Helvetica Regular and Helvetica Condensed are two styles of the typeface Helvetica; helvetica-regular.otf and helvetica-condensed.otf are two font files in the same font family.
script A writing system. English and Dutch both use Latin script; Cyrillic and Arabic are two other common scripts.
subsetting The process of removing glyphs and features you don’t need from a font, mainly to reduce the file size. This is usually done by removing scripts you don’t need.
rendering When an operating system displays glyphs on a screen. Mac OS renders fonts exactly as they were designed, while Windows tries to make them more readable. Text usually appears thicker on Mac. See Statistics for information on OS market share.

Paragraph properties

The optimal line length for websites is 50–75 characters.

Paragraphs should have no more than 3–4 sentences apiece.

Font styles

Sans-serif

This is a sans-serif font.

These are best for screens and other digital media, with their clean lines and pixel-friendly glyphs.

Serif

This is a serif font.

These are best for print and other text-heavy media such as ebooks.

Serif fonts have a traditional, old-school feel to them, and are easier to read for long periods on paper and e-ink screens. Examples include Times New Roman, Georgia, Rockwell, and Garamond.

  • Old-style: originated in 1465, and had great readability on the rough paper of the time. These fonts have small differences between thin and thick lines, and curves connecting the serifs to the letter strokes. These are most commonly used in print media.
  • Transitional: originated in the 1700s. They have a larger idfference between thin and thick lines, with more subtle, tapered serifs.
  • Didone: originated in the late 1700s to highlight improvements in printing technology. Serifs are typically very narrow and sharp with a constant width, giving the fonts a sleeker look. They also typically have heavier vertical strokes than previous serif fonts. Didone fonts are the most used serif fonts in web design.
  • Slab: originated in the early 1800s. They typically have a constant line thickness with pronounced serifs, clean edges, and sharp corners.

Monospace

This is a monospace font.

This is most frequently used to represent code, for headings, or as a stylistic choice.

Cursive/Script

This is a cursive/script font.

These give a handwritten feel to text, either to inject a sense of fun or elegance.

Fantasy

This is a fantasy font.

These are typically used for titles, main headings, or other standalone text that needs to be stylistically different from the rest of the page.
