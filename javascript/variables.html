Declaring variables

ES6 JavaScript introduces two new declaration keywords: const and let.

Keyword Scope Usage Examples
var Global Allows declaration of mutable variables. New variables declared locally with the same name will overwrite global variables’s values. Avoid using var; use const or let instead.
const Block

Allows declaration of immutable (constant) block-scope variables. New variables declared locally with the same name will not overwrite global variables’s values. This is best for variables that don’t need to be updated or re-declared. const declarations must be initialized when declared.

Note that while objects declared with const cannot be updated, the properties and child objects within it can.
  • Data structures
  • Dates and times
let Block Allows declaration of mutable block-scope variables. New variables declared locally with the same name will not overwrite global variables’s values. This is best for variables whose values need to change based on program flow. let declarations do not need to be initialized with a value when declared.
  • Counters and loop iterators
  • Results of logic evaluations (true/false with different values depending on outcome)

Assigning values and performing arithmetic

To assign or increment a variable, use the following commomly used operators. The full list of assignment operators can be found here.

Assignment operators

Operator Description Usage Example
= Assignment Sets a variable equal to a value x = 5;
+= Addition assignment Increments a variable by a value, or concatenates strings together x += 5;
x += 'string';
-= Subtraction assignment Subtracks a value from a variable’s value (only works on numbers) x -= 5;
*= Multiplication assignment Multiples a variable by a value (only works on numbers) x *= 5;
/= Division assignment Divides a variable by a value (only works on numbers) x /= 5;
%= Remainder assignment Assigns a variable the remainder of it divided by a value let a = 3;
a %= 2; // outputs 1
** Exponentiation assignment Multiples a variable by itself a specified number of times (sets its exponent) let a = 3;
a ** 2; // outputs 9

Arithmetic operators

Operator Description Usage Example
% Remainder Returns the integer remainder of dividing two values x = 12 % 5; // sets x to 2
++ Increment Adds 1 to the value of an operand x++;
-- Decrement Subtracts 1 from the value of an operand x--;
- Unary negation Returns the negation of an operand let x = 3; return (-x); // returns -3
+ Unary plus Attempts to convert the operand to a number if it isn’t one already +"3" // returns 3
+true // returns 1
** Exponentiation operator Calculates the base to the exponent power 2 ** 3 // returns 8

Setting default values

If a value input into a variable is null, you can set a default value to display instead.

let username = usernameInput || 'Default value';

This is known as short-circuit evaluation and allows for failsafes in case of invalid or null input.

Data types

Simple data types (number, boolean, null, undefined)

Numbers can be any number, either a whole number (integer) or with decimal points (float).

Booleans are a binary true or false, usually used as an indicator flag or an on/off switch.

Null variables have the intentional absence of a value; the variable exists but does not have a valid value. Example: let x = null;

Undefined variables have not yet been declared, or have not had a value defined. Example: let x;

Strings

Strings are how plain text or other simple non-numerical values are stored. For example, 'foobar' is a string instead of a number, array, etc.

Enclose strings in 'single quotes', "double quotes", or `backticks`; it doesn’t matter what method as long as it’s consistent. Single quotes and backticks are best when generating HTML code via JavaScript, to avoid having to escape double quotes. Backticks are used for template literals; more on this in a bit.

'This is a string.'
"This is also a string."
`And this is another string.`

To escape a character in a string and prevent it from terminating the string early, use a \ before the character.

'Use a backslash to make sure that any use of the same character doesn\'t break your string.'

Strings can be concatenated (added) with the + operator. When constructing strings of HTML code containing variables, however, this can get messy and lead to mistakes; it’s best to use template literals instead. Enclose variables and operations in ${curly braces} with a dollar sign preceding them.

`Use template literals to add in ${variables} directly to a string.`;

Arrays

Arrays are best used to hold sets of data that need to remain in a specific order. Their data can be accessed by numerical index, starting at 0.

To get the size of an array, use arrayName.length.

Methods

Arrays come with default methods attached to them, to perform basic operations. Use these methods with arrayName.method().

Method Usage
.push([data]) Adds data to the end of an array
.pop() Removes and returns the last element of an array
.unshift([data]) Adds data to the beginning of an array
.shift() Removes and returns the first element of an array
.slice([start], [end]) Copies the indicated range of an array into a new array; end is not included and the operation proceeds only up to end - 1. The original array is not modified.
.isArray([variable]) Returns true if the variable contains an array, false otherwise
.every([function]) Returns true if every element in the array satisfies the function
.filter([function]) Returns a new array containing all elements that satisfy the filtering function
.findIndex([function]) Returns the index of the first element where the function returns true
.forEach([function]) Executes a function for each element in the array; more on forEach() loops here
.join() Joins all array elements together into one string
.map([function]) Executes a function for each element in the array, and returns a new array of the results
.reverse() Reverses the elements of an array in place; first becomes last, and vice versa
.some([function]) Returns true if at least one element in the array satisfies the function
.sort() Sorts the elements of an array in place

Objects

Objects are collections of related data organized into key-value pairs. They are similar to arrays, except values can be accessed by name using dot notation: object.key1, and their order within the object may change without warning. For keys with spaces, variable names as keys, or key names that are enclosed with quotes, use bracket notation: object['key 2']. Access keys of child objects using object.childname.keyname.

const objectName = {
     key1: value1,
     'key 2': 'value 2'
};

Object properties can also be accessed with bracket notation: object[property], and this is most useful when iterating through an object. For example:

for (const propertyName in objectName) {
     let property = objectName[propertyName];
     [code to execute for each iteration]
}

To remove a key or child object from an object, use delete objectName.key.
