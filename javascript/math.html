Properties

The Math object has a handful of built-in properties to allow you to easily access commonly used mathematical constants.

Property Usage Output
Math.E Euler’s constant and the base of natural logarithms; approximately 2.718
Math.LN2 Natural logarithm of 2; approximately 0.693
Math.LN10 Natural logarithm of 10; approximately 2.303
Math.LOG2E Base-2 logarithm of E; approximately 1.443
Math.LOG10E Base-10 logarithm of E; approximately 0.434
Math.PI π: ratio of a circle’s circumference to its diameter; approximately 3.14159
Math.SQRT1_2 Square root of ½; approximately 0.707
Math.SQRT2 Square root of 2; approximately 1.414

Methods

The Math object also has several methods built into it to allow you to easily perform common mathematical operations without having to custom-code helper functions.

This is not an exhaustive list; there are several other Math methods for things like trignometric functions and binary operations.

Method Usage Example Output
Math.abs(x) Returns the absolute value of x. Math.abs(-10)
Math.cbrt(x) Returns the cube root of x. Math.cbrt(8)
Math.ceil(x) Returns the smallest integer greater than or equal to x. Math.ceil(3.14)
Math.floor(x) Returns the largest integer less than or equal to x. Math.floor(3.14)
Math.max([parameters]) Returns the largest of the values passed to it, or NaN if any of the parameters is not a number and can’t be converted into one. Math.max(1, 3, 2)
Math.min([parameters]) Returns the smallest of the values passed to it, or NaN if any of the parameters is not a number and can’t be converted into one. Math.min(1, 3, 2)
Math.pow(x, y) Returns base x to the exponent power y (that is, xy). Math.pow(2, 3)
Math.random()

Returns a pseudo-random number between 0 and 1 (inclusive of 0 but exclusive of 1). To generate a random value within a range, multiply by the range’s maximum value. Use Math.floor(...) or Math.ceil(...) to round to the nearest integer.

Note: Math.random() does not provide cryptographically secure random numbers, as this method only returns a pseudo-random number.

Math.round(x) Returns the value of the number x rounded to the nearest integer. Math.round(3.2)
Math.round(3.6)
Math.sign(x) Returns the sign of the x, indicating whether x is positive, negative, or zero. Math.sign(-4)
Math.sign(0)
Math.sign(4)
Math.sqrt(x) Returns the positive square root of x. Math.sqrt(9)
Math.trunc(x) Returns the integer portion of x, removing any fractional digits. Math.trunc(3.14)
