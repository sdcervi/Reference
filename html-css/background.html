Individual properties

Property Value(s) Usage
background‑image url("path‑to‑image.jpg"); Allows you to set any image as a background. Make sure it has sufficient resolution, is positioned correctly, and follows accessibility guidelines by having sufficient contrast with any text overlaid on top of it and not being too visually busy.
background‑repeat repeat: Repeats on both X and Y axes
repeat‑x: Repeats on just the X axis
repeat‑y: Repeats on just the Y axis
space: Repeats as many times as possible without clipping, pins the outside repeats to the edges of the element, and evenly distributes space between repeats
round: Functions like space but stretches the image to fit the empty spaces
no‑repeat: Will not repeat on any axis		 Allows you to change how and whether a background will repeat within its container
background‑attachment fixed: The background will stay fixed relative to the viewport, and foreground content will scroll over it
scroll: The background is fixed relative to its container, will not scroll with its foreground content, but will scroll with the container
local: The background will scroll with its foreground content within its container		 Changes the scrolling behavior of the background relative to its container and foreground content
background‑position top, right, bottom, left, center: Keywords to fix a background to a particular border of its container, or center it within the container. Will set the other axis to 50% to center it on the edge specified. Two keywords can be used to define both X and Y position. Default value is top left.
[length] or [percentage]: Specifies the X-coordiate relative to the left edge, with the Y-coordinate set to 50%. Default value is 0 0.
[keyword] [offset]: Defines a length offset for the preceding keyword. If a keyword has no length value following it, the length defaults to 0.		 Allows for adjustment of a background image’s position within its container
background‑size contain: Scales the image proportionally to fit within the container
cover: Stretches the image proportionally to completely fill the container, and crops the image so that no empty space is left		 Changes how and if a background is scaled to fit or fill the container
background‑color Hex value, keyword, RGB, RGBa, HSL, HSLa Sets a solid background color; this is behind the background-image layer and can be used as a fallback if the image fails to load. It will be visible through any transparent pixels in the background image.

Shorthand

The above properties can (mostly) be condensed into one shorthand definition. background-size must immediately follow background-position and have a slash between them.

background: [background-image] [background-position] / [background-size] [background-repeat] [background-attachment] [background-color];
