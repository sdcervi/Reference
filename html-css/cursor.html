Cursors

  • Home

Different usability patterns and elements within a page need different cursors, to help users identify how to interact with a page. For example, the buttons below have different cursor settings. One changes the cursor to a hand to show the user it’s clickable; the other doesn’t. It’s considered best practice to use the first usability pattern.

Use the CSS rule cursor: [value] to define a specific cursor for an element or one of it’s states.

No cursor is shown. This has a very limited set of use cases, since having the cursor disappear without warning can alarm users.

One applicable use case is displaying a web page for viewing only, such as at a museum as part of an installation or exhibit.
