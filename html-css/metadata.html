Metadata

  • Home

The meta element goes in the head of an HTML document, and contains information about the page that can be used by search engines, screen readers, and other systems.

Format: <meta name="property" content="value" />, or <meta http-equiv="property" content="value" />.

Attribute Type Content
description name This tells search engines what the page is about. Maximum 155 characters, and often displayed under the page name and URL in search results.
keywords name Comma-separated keywords to help users find your site. However, with modern page ranking algorithms, these keywords have much less effect than they used to.
robots name Indicates whether search engines should include this and/or any child pages in their results.
noindex tells a search engine not to add the page to its results.
nofollow tells a search engine to add this page but not any pages this one links to.
author http-equiv This defines the author of the webpage.
pragma http-equiv This tells the browser whether to cache (cache) the page or not (no-cache).
expires http-equiv This tells the browser when the page expires and should no longer be cached. Dates must be specified in the format Fri, 04 Apr 2021 18:36:43 GMT.
