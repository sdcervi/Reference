Box model components

Every content box around an element has properties that affect its final size and appearance. If a width and/or height is specified for the content box, then the padding, borders, and margin are all added to that, which can make an element take up more space on a page than expected.

Property Usage Color in diagram
height and width These, along with min-height and min-width, define the base size of the content box.  
padding Padding is the space between the content box and the border.  
border Borders are rendered between padding and margin.  
margin Margin sits outside of everything else.

Box model sizing options

By default, modern browsers use the content-box method, as shown above, which adds padding and borders on the outside of the defined element’s size. However, by setting box-sizing: border-box; you can tell the browser to include any width/height, padding, and borders into the calculation of the total element size.
