What are CSS functions?

CSS functions are predefined logic blocks that can be reused for simple calculations and operations on CSS values. They drastically expand how styling can be applied; for example, setting a font size to a calculated value that uses the page width and a defined size.

Functions are used as part of a value declaration; some take multiple comma-separated arguments.

selector {
     property: function-name(argument);
}

CSS transforms are one kind of function. Other functions include the following; this isn’t an exhaustive list but rather just commonly used ones.

Math functions

Function Usage Example
calc([expression]) Allows you to perform calculations via a mathematical expression. This can be used anywhere a length, frequency, angle, time, percentage, or number is expected. Surround the mathematical operators with spaces to avoid a syntax error. width: calc(50% - 50px);
clamp([min], [ideal], [max]) This clamps a value (ideal) between an upper (max) and lower (min) bound. The final value will never be less than min or greater than max, and will be ideal when possible. This can be used anywhere a length, frequency, angle, time, percentage, or number is expected. Separate specified values with commas. width: clamp(100px, 20%, 800px);
max([values]) Takes one or more comma-separated expressions and returns the largest value from that list. This can be used anywhere a length, frequency, angle, time, percentage, or number is expected. width: max(40vw, 5em, 300px);
min([values]) Takes one or more comma-separated expressions and returns the smallest value from that list. This can be used anywhere a length, frequency, angle, time, percentage, or number is expected. width: min(40vw, 5em, 300px);

Filter functions

Function Usage Example Result
blur([radius]) Applies a Gaussian blur to an element. Larger values will have more blur. filter: blur(2px);
brightness([value]) Applies a linear multiplier to an element, making it look brighter or darker. A value over 100% or 1 will brighten the element, while a value under 100% or 1 will darken it. 0 will turn an element completely black. filter: brightness(200%);
contrast([value]) Adjusts the contrast of an element. A value over 100% or 1 will increase the contrast, while a value under 100% or 1 will decrease it. 0 will turn an element completely gray. filter: contrast(0.3);
drop-shadow([x-offset] [y-offset] [radius] [color]) Applies a drop shadow to an element. This works with images and elements that have transparency within their bounding box, such as PNGs and SVGs. This is not to be confused with box-shadow, which applies a shadow to the element’s bounding box rather than the element itself. filter: drop‑shadow(3px 3px 1px gray);
grayscale([value]) Converts an element to grayscape. A value of 100% or 1 will turn an element completely grayscale, while a value under 100% or 1 will decrease the saturation. 0 will keep the element fully colored. filter grayscale(80%);
hue-rotate([value]) Rotates the hue of an element. Values can be in angles, radians, or turns. A full turn (360deg, 1turn, 6.28rad) will be equivalent to 0 and have no effect. filter hue‑rotate(180deg);
invert([value]) Inverts the colors of an element. A value of 100% or 1 will fully invert the colors, while a value of 0% or 0 will have no effect. 0.5 or 50% will turn an element completely grayscale. filter: invert(1);
opacity([value]) Sets the transparency of an element. A value of 100% or 1 will be fully opaque, while a value of 0% or 0 will have be fully transparent. filter opacity(0.3);
saturate([value]) Hyper-saturates or desaturates an element. A value under 100% or 1 will desaturate the element, while a value over 100% or 1 will hyper-saturate it. 0 will fully desaturate the element, while 100% or 1 will leave the element unchanged. filter: saturate(0.3);
sepia([value]) Applies a warm, yellow-brown filter to an element. A value of 100% or 1 will fully apply the filter, while a value of 0% or 0 will have no effect. filter sepia(1);

Color functions

These can be used anywhere a color value is valid. More information on color spaces »

Function Usage Example Result
hsl([hue], [saturation], [lightness]) Specifies a color by its hue, saturation, and lightness. hsl(225, 100%, 50%)
hsla([hue], [saturation], [lightness], [opacity]) Specifies a color by its hue, saturation, lightness and alpha (opacity). hsla(225, 100%, 50%, 0.5)
rgb([red], [green], [blue]) Specifies a color by its hue, saturation, and lightness. rgb(74, 133, 18)
rgba([red], [green], [blue], [opacity]) Specifies a color by its hue, saturation, lightness and alpha (opacity). rgba(74, 133, 18, 0.5)

Image functions

These can be used wherever an image is valid as the value for a property.

Function Usage Example Result
conic-gradient(from [angle] at [x-center] [y-center], [color1] [angle-start] [angle-stop], [color2] [angle-start] [angle-stop], ...) Creates a gradient with color transitions rotated around a defined center point, rather than radiating outward in rings. Use the optional from keyword to denote the angle that the gradient starts at, and at to set the center point. Color stops then follow, with optional angles for each stop. background: conic-gradient(from 0.25turn at 50% 30%, #f69d3c, 10deg, #3f87a6, 350deg, #ebf8e1);
linear-gradient([starting point] [angle], [color stop 1], [color stop 2], ...) Creates a gradient with color transitions in a linear direction. Read the MDN docs for available keywords and syntax. background: linear-gradient(0.25turn, #3f87a6, #ebf8e1, #f69d3c);
radial-gradient([shape] at [position], [color stop 1], [color stop 2], ...) Creates a gradient that radiates outward from a center point in rings. Read the MDN docs for available keywords and syntax. background: radial-gradient(#e66465, #9198e5);

Shape functions

These are used for defining shapes for various CSS properties such as clip-path, offset-path or shape-outside. Read the documentation on these functions here.

Function Usage Example
circle([radius] at [x-center] [y-center]) Defines a circular path clip-path: circle(6rem at 12rem 8rem);
ellipse([x-radius] [y-radius] at [x-center] [y-center]) Defines an ellipse path shape-outside: ellipse(40% 50% at left);
inset([offset-top] [offset-right] [offset-bottom] [offset-left] [border-corner-shape] [border-radius]) Defines an inset rectangle path shape-outside: inset(20px 50px 10px 0 round 50px);
polygon([fill rule], [corner1], [corner2], [corner3], ...) Defines a polygone path clip-path: polygon(50% 2.4%, 34.5% 33.8%, 0% 38.8%, 25% 63.1%, 19.1% 97.6%);
path([fill rule], [string]) Defines a shape based on an SVG path string path("M 10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80");

Reference functions

These are used to reference values defined elsewhere.

Function Usage Example
attr([value] [type], [fallback]) Retrieves the value of an attribute of the selected element and uses it in the stylesheet. MDN documentation here width: attr(data‑width px, inherit);
env([value], [fallback]) References an environment variable, for example with use for a desktop Progressive Web App. MDN documentation here margin‑top: env(safe‑area‑inset‑top, 20px);
url([value]) Used to include a file in a CSS property’s value, i.e. as a background image, font, or external stylesheet. The URL can be wrapped in single or double quotes as needed, if it includes special characters. background‑image: url(https://example.com/images/myImg.jpg);
var([value]) Used to reference a CSS variable (custom property) border‑color: var(‑‑color‑a);
